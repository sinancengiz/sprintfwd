# README

This is the sprintFWD project.

Clone the project:

* git clone https://github.com/sinancengiz/sprintfwd.git

Bundle install

* bundle install

Create Database
* rails db:create
* rails db:migrate
* rails db:seed


Run Server
* rails s

Navigate to api-docs in local host
* http://127.0.0.1:3000/api-docs/index.html
